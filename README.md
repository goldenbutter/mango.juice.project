# 🥭 Raw Pressery | Alphanso Mango

https://github.com/goldenbutter/mango.juice.project/raw/main/Final_Output.mp4

A premium, highly interactive single-page landing website. This project seamlessly integrates a massive **200-frame 3D animation sequence** directly into the background of the page using HTML5 `<canvas>`. The 3D scene rotates dynamically as you scroll down the page or drag your mouse across the screen! 

## ✨ Features

- **🎥 Full-Bleed 3D Background Canvas:** The custom JS algorithm draws the sequence to completely fill any screen size, acting exactly like CSS `object-fit: cover` to avoid stretching or black bars.
- **📜 Scroll-to-Rotate Interaction:** Map your scroll progression across a massive 500vh container directly to the 200 frames of the animation. 
- **🖱️ Drag-to-Rotate Interaction:** Click and drag your mouse left or right across the bottle to interact with the 3D product manually. Wraps infinitely 360 degrees.
- **🧈 Butter-Smooth Easing (Lerp):** An optimized `requestAnimationFrame` loop mathematically smooths out the rotation (using linear interpolation) so the animation feels incredibly premium, even if you are using a chunky scroll wheel.
- **🌟 Mouse Parallax Effect:** Gently tracks mouse movement to slightly shift the background typography and foreground elements, simulating 3D depth of field. 
- **🚀 Ultra-lightweight:** 100% Vanilla JavaScript and zero heavy frontend frameworks. Styled instantly with Tailwind CSS.

## 🛠️ Built With
* **HTML5 `<canvas>`** - For high performance 3D image sequencing.
* **Vanilla JavaScript** - For interactions, easing, and canvas rendering.
* **Tailwind CSS** (via CDN) - For rapid, responsive UI styling.

## 🎨 AI Asset Workflow (How the frames were generated)

If you'd like to recreate the 200 sequence frames yourself without touching any heavy 3D software:
1. **Base Generation:** The main 3D-style composition was generated from scratch using the **[Google FX Whisk](https://labs.google/fx/tools/whisk/project)** tool with the following prompt:
```
   A hyper-cinematic, high-speed macro composition capturing the dramatic assembly and collision of premium mango elements. A vibrant, saturated orange gradient forms a seamless void backdrop. From the edges of the frame, a dynamic vortex of thick, glossy Alphonso mango purée spirals inward, intertwining violently yet gracefully with jagged, velvety cubes of ripe fruit. 
   Millions of microscopic, backlit nectar droplets hang suspended mid-air, refracting golden light like liquid amber. 
   At the center, a RAW Pressery Alphonso Mango bottle materializes from the swirling puree, its surface glistening with heavy condensation. Perfectly ripe mango chunks, lush green leaves, and crystal-clear juice droplets float into a balanced, suspended composition around the bottle. 
   Lighting transitions from soft ambient to a dramatic, volumetric golden-hour glow, emphasizing the pulpy, fibrous grain of the fruit, the surface tension of the juice, and the glossy texture of the puree. The camera performs a subtle, slow push-in. Rendered in ultra-crisp 8K, slow motion, with extreme clarity and commercial-grade detail.
   ```
2. **Video Animation:** That static base image was animated into a high-quality MP4 directly within the same Whisk project.
3. **Sequence Extraction:** The MP4 was uploaded to **Ezgif** and converted into an image sequence at **25 FPS**, instantly generating the 200 beautifully timed `.jpg` layers used on this canvas!

## 🚀 How To Run Locally

Because this website rapidly preloads & draws 200 local image files onto a canvas element, strict modern browser security (CORS) will block the animation if you simply double-click the `index.html` file. 

**You MUST run it via a local web server.** 

### Option 1: Using Python
If you have Python installed, open your terminal in this project directory and run:
```bash
python -m http.server 3000
```
Then visit: `http://localhost:3000`

### Option 2: Using Node.js
If you have Node/npm installed, you can use the `serve` package:
```bash
npx serve -p 3000
```
Then visit: `http://localhost:3000`

### Option 3: Using VS Code
1. Install the **Live Server** extension in Visual Studio Code.
2. Right-click on `index.html` inside the explorer.
3. Select **Open with Live Server**.
